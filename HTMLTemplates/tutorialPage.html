Tutorial

What will you be doing today?

In this study, we aim to investigate the differences between using a conventional computer mouse and a gesture-based method (using the Leap Motion, an optical hand-tracking device) to move a cursor around the screen. To accomplish this, we have prepared a set of forms that you will need to fill out. To complete the forms, you will need to navigate to the correct fields with your cursor, enter the required text using the keyboard, and move on to the next field.

The flow of the user study is as follows:

  • You will be given time to familiarize yourself with the gesture input system.
  • Depending on your user ID, you will start by using the computer mouse or the gesture system for cursor movement.
  • You will fill out a form and an email with predetermined sample values.
  • You will repeat those same tasks with the other input device you have not yet used.

Entering text

You will be expected to enter text in certain fields. When the field is ready, you will see light gray "placeholder text" within the field. Once you have navigated to and selected the field, you will need to enter into the field the placeholder text verbatim.

You will receive visual feedback as you enter text into the field. If your input so far is correct, your text will appear as black. (The goal is effectively to turn all the light gray text into black text). If you have made a mistake, your text will turn red and the light gray text will disappear. To correct the mistake, press backspace until your text becomes black again and the light gray text reappears. When you have entered the text fully, it will turn green and the field will be disabled, indicating that it is time to proceed to the next field. You may experiment with this below.

Moving the cursor

During the user study, you will see a bar at the top of the page (see Fig. 1) that indicates which input device that is currently active, and the task you are currently working on. This active input device is the device that you are expected to be using to move the cursor. Take care in ensuring that you are using the correct device. When you are expected to use the mouse, gesture input is disabled. However, when you are expected to use gesture input, the mouse is still operational, because it is impractical to disable the mouse entirely.

Moving the cursor with the mouse

Cursor movement using the mouse works in the same way that you would expect it to.

Moving the cursor with hand gestures

Cursor movement using the gesture input system requires some explanation. The Leap Motion is placed in front of your keyboard. Note that only one hand may control the cursor at a time. To move the cursor, you hand must be placed in a "thumbs up" position (see Fig. 2). In this position, the mouse will move in the direction that your fingertips are pointing. (Your thumbs do not necessarily need to be pointing directly up, but your hand should be posed in such a manner that allows you to do so).

Note that this pose only allows the fingertips to face any direction within a half-circle. As a result, you will have to use both hands to move the cursor. If you wanted to move to the cursor to the left, you would use your right hand. Similarly, if you wanted to move the cursor to the right, you would use your left hand. The direction of your palm indicates the "direction" of the half-circle you can move your cursor in.

To perform a left click, simply face your palm downwards.

On the other screen, you should be able to see a visualization of what the Leap Motion is seeing. Gesture input is currently active. Feel free to experiment with moving the mouse. A reasercher is available if you need help with the gesture input system. Once you are comfortable with the gesture input system, you may begin the user study.
