Tutorial

Summary of the user study

In this study, we aim to investigate the differences between using a conventional computer mouse and an optical hand-tracking device to move a cursor around the screen. To accomplish this, we have prepared a set of forms that you will need to fill out. To complete the forms, you will need to navigate to the correct fields with your cursor, enter the required text using the keyboard, and move on to the next field.

The flow of the user study is as follows:

  • You will be given time to familiarize yourself with the inputs expected of you.
  • Depending on your user ID, you will start by using the mouse or the Leap Motion optical hand-tracker for cursor movement.
  • You will fill out a form and an email.
  • You will repeat those tasks with the input device you have not yet used.

Explanation of text entry

You will be expected to enter text in certain fields. When the field is ready, you will see light gray "placeholder text" within the field. Once you have navigated to and selected the field, you will need to enter into the field the placeholder text verbatim.

You will receive visual feedback as you enter text into the field. If your input so far is correct, your text will appear as black. (The goal is effectively to turn all the light gray text into black text). If you have made a mistake, your text will turn red and the light gray text will disappear. To correct the mistake, press backspace until your text becomes black again and the light gray text reappears. When you have entered the text fully, it will turn green and the field will be disabled, indicating that it is time to proceed to the next field.

Explanation of cursor movement

During the user study, you will see a bar at the top of the page that indicates which task you are currently working on, and the input device that is currently active. This is the device that you are expected to be using to move the cursor. Take care in ensuring that you are using the correct device. When you are expected to use the mouse, gesture input is disabled. However, when you are expected to use gesture input, the mouse is still operational, because it is impractical (and possibly dangerous) to disable the mouse entirely.

Cursor movement using the mouse is exactly the same as you would expect.

Cursor movement using the Leap Motion (gesture input) requires some explanation. The core principal behind the gesture-based mouse movement system is as follows: when the user's hands are in a specific pose, the cursor will move at a constant velocity in the direction that the user's fingertips are pointing. The specific pose in this case is the "thumb up" pose: the user's hand should be oriented in such a way that their thumb is pointing directly upwards, as if they were giving a "thumbs up" gesture. (Your thumbs need not be pointing directly up, but your hand should be posed in such a manner that allows you to do so).

Note that this pose only allows the fingertips to face any direction within a half-circle. This is problematic: we are only able to move the cursor in 180 degrees (as opposed to the expected 360 degrees). This is alleviated by using both hands to move the cursor. If you wanted to move to the cursor to the left, you would use your right hand. Similarly, if you wanted to move the cursor to the right, you would use your left hand. The direction of your palm indicates the "direction" of the half-circle you can move your cursor in. As a result, you should only be using one hand at a time to control the cursor.

To click, simply face your palm downwards.

On the other screen, you should be able to see a visualization of what the Leap Motion is seeing. Gesture input is currently active. Feel free to experiment with moving the mouse. The individual that is running the user study is available if you need help with the gesture input system. Once you are comfortable with the gesture input system, you may begin the user study.
